# COSMOEYE

# 🛰️ CosmoEye-DRR: Cosmic Orbital Surveillance and Monitored Observation Eye for Disaster Risk Reduction

CosmoEye-DRR is a self-contained, multi-modal deep learning pipeline built **completely from scratch without external cloud APIs**. The system is engineered to bridge the "last-mile" communication gap during monsoon disasters in Nepal by converting raw, multi-spectral satellite imagery directly into actionable, plain-text emergency bulletins for localized rescue teams.
